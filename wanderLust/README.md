🏡 Wanderlust
Wanderlust is a full-stack web application that allows users to explore and create travel listings.
It provides an interactive platform where users can search for homes at various locations or become a host.

✨ Features
-🏠 Browse Listings – Explore various travel stays with details and images.
-📍 Map Integration – View locations geographically.
-➕ Create Listings – Add new places with descriptions, pricing, location and images.
-✏️ Edit & Delete Listings – Manage your own listings easily.
-⭐ Reviews and ratings system - Share your experience with others. 

⚙️Implementation details ->
- Implemented a full CRUD system for managing listings
- Used Passport.js for authentication
- Applied server-side validation using Joi
- Used Cloudinary for image uploads

🛠️ Technologies
Frontend: HTML, CSS, JavaScript, Bootstrap, EJS 
Backend: Node.js , Express
Database: MongoDB
Maps & APIs: Mapbox
Other Tools: Cloudinary , Joi , Passport.js
